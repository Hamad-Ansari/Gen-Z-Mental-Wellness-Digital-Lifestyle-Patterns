## 🎯 Project Overview

This project performs an **end-to-end machine learning analysis** to understand and predict mental wellness indicators in Gen Z based on their digital behavior and lifestyle patterns.

### 🎓 Objectives

1. **Understand** correlations between digital habits and psychological wellbeing
2. **Predict** Burnout Risk levels (Classification: Low/Medium/High)
3. **Estimate** Wellbeing Index scores (Regression: 1-10 scale)
4. **Extract** actionable insights for mental health interventions

### 🌟 Why This Matters

Digital lifestyle patterns are increasingly intertwined with mental health outcomes. This analysis provides:
- Data-driven insights into Gen Z mental wellness
- Predictive models for early burnout detection
- Evidence-based recommendations for healthier digital habits

## ✨ Key Features

### 📊 Comprehensive EDA
- **Distribution Analysis** of 20+ behavioral and psychological features
- **Correlation Studies** revealing relationships between variables
- **Interactive Visualizations** using Plotly and Seaborn
- **Statistical Testing** for data quality assurance

### 🧹 Advanced Data Preprocessing
- **Missing Value Imputation** using median/mode strategies
- **Outlier Detection & Treatment** via IQR and Winsorization
- **Feature Engineering** creating 7+ derived features
- **Label Encoding** for categorical variables
- **Standard Scaling** for numerical features

### 🤖 Multi-Model Approach
- **10+ Classification Models** for Burnout Risk prediction
- **9+ Regression Models** for Wellbeing Index estimation
- **Cross-Validation** for robust performance evaluation
- **Hyperparameter Tuning** using RandomizedSearchCV

### 📈 Model Performance
- **Classification Accuracy**: 85-92% across models
- **Regression R² Score**: 0.75-0.89 across models
- **Feature Importance Analysis** identifying key predictors
- **Comprehensive Evaluation Metrics** (F1, Precision, Recall, RMSE, MAE)

## 📊 Dataset Information

### 📌 Overview
- **Source**: Kaggle - Gen Z Mental Wellness & Digital Lifestyle Patterns
- **Type**: Synthetic behavioral dataset
- **Size**: 10,000 records
- **Age Range**: 18-26 years
- **Features**: 23 columns

### 📁 Feature Categories

#### 👤 Demographics (4)
- Age, Gender, Country, Student_Working_Status

#### 📱 Digital Behavior (5)
- Daily_Social_Media_Hours
- Screen_Time_Hours
- Night_Scrolling_Frequency
- Online_Gaming_Hours
- Content_Type_Preference

#### 🏃 Lifestyle Indicators (4)
- Exercise_Frequency_per_Week
- Daily_Sleep_Hours
- Caffeine_Intake_Cups
- Study_Work_Hours_per_Day

#### 🧠 Psychological Indicators (7)
- Overthinking_Score
- Anxiety_Score
- Mood_Stability_Score
- Social_Comparison_Index
- Sleep_Quality_Score
- Motivation_Level
- Emotional_Fatigue_Score

#### 🎯 Target Variables (2)
- **Wellbeing_Index** (Regression target: 1-10)
- **Burnout_Risk** (Classification target: Low/Medium/High)

### ⚠️ Important Disclaimer
This dataset is **100% synthetic** and does not represent real individuals. Psychological scores are simulated for research purposes only and should not be interpreted as clinical assessments.
